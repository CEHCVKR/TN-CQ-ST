## Folder guide

### 1) EXPERIENCE (INTERVIEW QA)

Purpose: interview preparation through previously asked questions and practical experiences.

Key files:

- [F2F Ninja Question.txt](EXPERIENCE%20%28INTERVIEW%20QA%29/F2F%20Ninja%20Question.txt)
- [F2F Prime Questions.txt](EXPERIENCE%20%28INTERVIEW%20QA%29/F2F%20Prime%20Questions.txt)
- [TCS NQT Questions.txt](EXPERIENCE%20%28INTERVIEW%20QA%29/TCS%20NQT%20Questions.txt)

### 2) NOTES

Purpose: aptitude and verbal/reasoning reference PDFs for exam preparation.

Examples:

- Numerical Ability
- Reasoning Ability
- Verbal Ability
- TCS NQT memory-based papers

Open folder: [NOTES](NOTES/)

### 3) Problems

Purpose: coding practice and solved question bank.

Main subfolders:

- [001](Problems/001/): topic-wise foundation problems
- [002](Problems/002/): additional coding sets and categorized problems
- [TCS NQT 2024 Coding Solutions](Problems/TCS%20NQT%202024%20Coding%20Solutions/): shift-wise solved questions

## Start here

If you are new to this repo, follow this order:

1. Read interview patterns in [EXPERIENCE (INTERVIEW QA)](EXPERIENCE%20%28INTERVIEW%20QA%29/)
2. Revise aptitude from [NOTES](NOTES/)
3. Practice basics from [Problems/001](Problems/001/)
4. Move to [Problems/002](Problems/002/)
5. Solve shift-specific sets in [TCS NQT 2024 Coding Solutions](Problems/TCS%20NQT%202024%20Coding%20Solutions/)
